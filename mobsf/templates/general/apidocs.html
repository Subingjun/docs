上传文件API

用于上传文件的API。支持的文件类型包括apk、zip、ipa和appx。

  • URL： /api/v1/upload

  • 方法： POST

  • 头部： Authorization:<api_key> 或 X-Mobsf-Api-Key:<api_key>

  • 数据参数：

参数名 参数值 必需
file multipart/form-data 是

  • 成功响应：

    • 状态码： 200
      内容类型： application/json; charset=utf-8
      内容： {"file_name": "diva-beta.apk", "hash": "82ab8b2193b3cfb1c737e3a786be363a", "scan_type": "apk"}

  • 错误响应：

    • 状态码： 500 Internal Server Error 或 405 Method Not Allowed 或 422 Unprocessable Entity
      内容类型： application/json; charset=utf-8
      内容： {"error": <错误消息>}

    或者

    • 状态码： 401 Unauthorized
      内容类型： application/json; charset=utf-8
      内容： {"error": "您未被授权进行此请求。"}

  • 示例调用：

    • curl -F 'file=@/Users/ajin/Desktop/diva-beta.apk' http://localhost:8000/api/v1/upload -H "Authorization:{{ api_key}}"

    或者

    • curl -F 'file=@/Users/ajin/Desktop/diva-beta.apk' http://localhost:8000/api/v1/upload -H "X-Mobsf-Api-Key:{{ api_key}}"

                  ["jakhar/aseem/diva/AccessControl3NotesActivity.java", "jakhar/aseem/diva/NotesProvider.java"]}
